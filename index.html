Hamburger Arrow React Native Component This is a React Native component called HamburgerArrow that implements a rotating hamburger menu icon. The icon transitions between an "arrow right" icon and a "menu" icon when pressed. It utilizes the React Native Animated API to animate the rotation of the icon.
